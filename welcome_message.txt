Ustasy - ваш автономный ИИ сотрудник (BETA)

На данный момент запускает все приложения на своём едином сервере
Позже вы сможете запустить его на своём личном Хостинге

                         
Пользовательские роли и их права в системе:

1. CREATOR

Это владелец системы, который имеет полный контроль над всем.
Права:

Выполнять любые команды в терминале
Читать, создавать, изменять и удалять любые файлы
Управлять пользователями и менять их роли
Перезапускать, останавливать или обновлять агента
Просматривать логи и системную информацию
Использовать все инструменты без ограничений
Игнорировать ограничения безопасности
Управлять API-ключами и переменными окружения
                         

2. ADMIN

Высокоуровневый администратор, который может управлять системой, но не может изменять роль Создателя.
Права:

Выполнять безопасные команды в терминале
Читать и редактировать файлы проекта
Перезапускать агента
Просматривать логи
Управлять пользователями (кроме Создателя)
Устанавливать пакеты Python
Запускать тесты и скрипты
                         

3. DEVELOPER

Может работать с кодом и файлами проекта, но не управляет пользователями и системными настройками.
Права:

Читать файлы
Создавать и редактировать файлы
Запускать скрипты проекта
Проводить тестирование
Использовать инструменты разработки
Не может выполнять опасные системные команды
Не может управлять пользователями
                         

4. USER

Обычный пользователь, который взаимодействует с агентом, но не может изменять системные файлы.
Права:

Задавать вопросы
Запрашивать генерацию кода
Создавать файлы в своей директории
Выполнять безопасные команды
Не может изменять основные файлы проекта
Не имеет доступа к системным файлам
Не может управлять другими пользователями
                         

5. GUEST

Очень ограниченный доступ.
Права:

Задавать вопросы
Читать разрешённые файлы
Не может создавать или редактировать файлы
Не может выполнять команды
Нет доступа к системе
